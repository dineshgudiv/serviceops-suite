# Security Policy

## Supported Versions
Only the latest `main` branch is supported for security fixes.

## Reporting a Vulnerability
Report vulnerabilities privately to: security@serviceops.local.
Do not open public issues for exploitable findings.

Include:
- Impact summary
- Reproduction steps
- Affected components and versions
- Suggested mitigations

## Response Targets
- Acknowledge report: within 2 business days
- Initial triage: within 5 business days
- Remediation plan: within 10 business days

## Disclosure
Coordinated disclosure is preferred after a fix is available.
